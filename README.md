# QR vCard Builder

A simple, free, browser-based QR code generator with logo support.

## Features

- Generate QR codes from a URL or vCard contact
- Add your logo in the center
- Rounded / blob-style QR design
- Download as PNG or SVG
- Copy SVG to clipboard
- No ads, no tracking, no server — everything runs locally in your browser

## Usage

Open `index.html` in any browser. No installation needed.
